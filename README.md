# SECTL Website

使用 Nuxt 4 构建的 SECTL 网站。

## 功能特性

- 响应式设计
- 深浅色主题切换
- 中英文双语
- SEO 优化
- 自动生成 sitemap
- GitHub 数据集成
- 现代化 UI

## 技术栈

Nuxt 4 + Nitro + Vite + Vue

## 开发

安装依赖：

```bash
pnpm install
```

启动开发服务器：

```bash
pnpm run dev
```

## 构建

生成静态文件：

```bash
pnpm run generate
```

预览静态文件：

```bash
pnpm run preview
```

## 部署

生成的静态文件位于 `.output/public` 目录，部署时指向该目录即可。
